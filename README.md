# Fingerprint-Based Attendance Management System

A biometric attendance solution integrating **Java**, **Arduino**, and a **centralized MySQL database** to automate and secure attendance tracking using fingerprint recognition.

---

## 🚀 Overview

This system replaces manual attendance processes with a fast, secure, and reliable fingerprint‑based workflow. It includes:

- Java Swing desktop application  
- Arduino fingerprint module integration  
- Centralized MySQL database  
- Attendance analytics and visualization  
- Secure login and user management  

---

## ✨ Features

- **Biometric Authentication** — Fingerprint scanning and verification using Arduino + sensor  
- **Java Desktop Application** — Clean UI built with Java Swing  
- **Centralized Database** — MySQL-based multi-user access  
- **Attendance Analytics** — Charts and visual summaries  
- **Hardware–Software Integration** — Real-time communication between Arduino and Java

## ⚙️ Setup & Installation

### **1. Database Setup**
- Install MySQL  
- Import the provided SQL schema  
- Update DB credentials in the Java config file  

### **2. Arduino Setup**
- Upload the `.ino` file  
- Connect fingerprint sensor  
- Test serial communication  

### **3. Run the Java Application**
- Open project in NetBeans/IntelliJ  
- Build and run the main class  
- Login using admin credentials
